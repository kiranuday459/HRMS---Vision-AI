13. Holiday Integration Flow
Admin-managed holidays integrated into leaves, balances & notifications
flowchart TD Admin["Admin"] --> AH["Add Holiday\nAdmin Dashboard"] AH --> DB[("Holiday Table in DB")] DB --> LC["Leave Day Calculation\nLeaveService"] DB --> NC["Notification Component\nCalendar Icon"] DB --> AR["All Roles View Holidays"] LC --> EW["Exclude Weekends"] LC --> EH["Exclude Public Holidays"] EW & EH --> AB["Accurate Leave Balance\nDeduction and Display"] NC --> HP["Holiday Calendar Popup"] HP --> UH["Display Upcoming Holidays\nfor All Users"] AR --> E["Employee — View Only"] AR --> R["Reporting Manager — View Only"] AR --> H["HR — View Only"] AR --> AV["Admin — Full Control"] style Admin fill:#553c9a,color:#fff style DB fill:#1e3a5f,color:#fff style AB fill:#1a936f,color:#fff style AV fill:#c05621,color:#fff
Key Design Decisions • Only Admins can create, edit, or delete holidays — all other roles have view-only access.
• Holidays are automatically excluded from leave day calculations, preventing incorrect balance deductions.
• The holiday calendar is accessible via the notification bell icon for all user roles.
• Leave balance deduction only counts actual working days (excluding weekends + holidays from the requested range).
